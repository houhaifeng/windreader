如果说今年会成为 Windows Phone 突围年，相信目前没人能够说不。
回顾 2013 年，微软持续加强 Windows Phone 系统功能，即将入门的诺基亚手机部门也不遗余力推出多种价位机型，上一季度诺记的 Windows Phone 出货量，预计将超过 1000 万部。
好事成双，微软此前宣布，去年 Q3 Windows Phone 的出货量已在全球 24 个国家超过 iPhone，而且在 14 个国家 Windows Phone 已成为第二大智能手机操作系统。另外 IDC 的数据对微软也非常利好：过去 12 个月 Windows Phone 出货量较往年上升 156%，升幅分别是 Android 与 iOS 的 3 倍和 6 倍。

而在欧洲，Windows Phone 表现也甚为迅猛，被形容为 EU5 的欧洲五国（英法德意西），消费者对 WP 设备有非常欢迎的态度。较前一年同期，2013 年 iOS 的市场份额从 20.8% 下降到 15.8%，而与此同时 WP 市占率却从 4.8% 提升到 10.2%，实现了突破双位数的里程碑。
从三强法则来看，Windows Phone 至少在欧洲已经有具备成为垄断移动系统市场 “三强”之一的条件。
数据来源：Kantar
Windows Phone 的销售业绩终于能够成为微软炫耀的资本，这貌似给人一种感觉：iOS 正受威胁，移动端操作系统的老二位置随时生变？
不过《福布斯》的看法却不一样。无可否认 Windows Phone 在 2013 年有了突破性的表现，但与 iPhone 相提并论依然为时尚早。个中原因有如下几个：

对于占有率较小的产品，获得更快的上升幅度是理所当然的；
（以上 24 个国家）除了意大利和诺基亚祖家芬兰以为，其余都是次发达国家，而 iPhone 在这些市场本来销量就有限；
Q3 对于苹果来说历来都是低迷季，原因很简单：消费者都在等新产品发布；
我们可以关注苹果即将公布的 Q4；
目前 Windows Phone 主力是 Lumia 510 和 520 两款低价机型，他们为整个阵营贡献了 42.4% 的出货量；
诺基亚占据了超过 93% 的 WP 份额，其他厂商估计没啥兴趣；

市场份额向 Windows Phone 倾斜，苹果会担心么？至少目前来说不会。因为目前微软仍未能改变 WP 设备低端化的状况。而 WP 设备争取到的市场空间，本来正正是苹果竞争力最薄弱也是最不关心的低端市场。因此，苹果一直关心的高端市场和利润率并没有因 Windows Phone 的成长而受到影响。如何让 Windows Phone 成功打入已被苹果、三星、索尼等厂商牢牢把控的高端市场，如何让消费者体验到 Windows Phone 高端机上的出色功能（如出色的成像水准），相信是微软面前依然是一大挑战。
但是已成事实的是，Windows Phone 已经成为不可忽视的存在。根据目前的数据，除了欧洲以外，澳洲很有可能成为 Windows Phone 下一个破十的市场。而随着系统和应用的日渐完善，对于价格敏感的中国市场，微软是否也能凭借性价比高的经济机型换来，更突破性的份额呢？毕竟只有两个操作系统的手机市场，还是太寂寞了。
